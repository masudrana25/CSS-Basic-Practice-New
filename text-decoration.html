          text-decoration-line (required)
          text-decoration-color (optional)
          text-decoration-style (optional)
          text-decoration-thickness (optional)
          
        ##text-decoration-line:
        
        Values: none, underline, overline, line-through, blink (This is non-standard and has limited browser support.)
        Purpose: Defines the type of text decoration to be applied.

        ##text-decoration-style:
        
        Values: solid, double, dotted, dashed, wavy
        Purpose: Defines the style of the text decoration line.

        ##text-decoration-color:
        
        Values: Any valid color value (color or rgba).
        Purpose: Specifies the color of the text decoration line.

        ##text-decoration-thickness (Currently in draft status, might not be widely supported yet):
        
        Values: auto, a length value.
        Purpose: Defines the thickness of the text decoration line.
